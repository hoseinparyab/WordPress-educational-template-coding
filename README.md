# Pishro WordPress Theme

![WordPress Logo](https://s.w.org/style/images/about/WordPress-logotype-standard.png)

![WordPress](https://img.shields.io/badge/WordPress-5.0+-blue.svg) ![PHP](https://img.shields.io/badge/PHP-7.4+-green.svg) ![License](https://img.shields.io/badge/License-GPL--2.0+-orange.svg)

A custom WordPress theme designed for educational purposes, featuring custom post types for TV shows and videos.

## Features

- Custom post types for TV and video content
- Responsive design with Owl Carousel integration
- Customizable header, footer, and page templates
- Integrated jQuery and custom JavaScript functionality

## Installation

1. Download the theme files.
2. Upload the `pishro` folder to your WordPress themes directory (`wp-content/themes/`).
3. Activate the theme from the WordPress admin dashboard under Appearance > Themes.

## Usage

- Customize the theme settings through the WordPress Customizer.
- Add TV and video content using the custom post types.
- Modify templates in the `template/` directory for specific layouts.

## File Structure

- `css/` - Stylesheets including Owl Carousel
- `js/` - JavaScript files
- `inc/` - Includes for custom post types and functionality
- `template/` - Custom page templates
- `fonts/` and `webfonts/` - Font files
- `img/` - Images

## Requirements

- WordPress 5.0+
- PHP 7.4+
